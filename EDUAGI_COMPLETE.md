## 🚀 How to Use It

### For Students:
```typescript
import { EducationalAGI } from './educational-agi-sdk';

const eduAGI = new EducationalAGI();

// Register
const me = eduAGI.registerStudent("Your Name", 10, 'visual');

// Create learning path
const path = eduAGI.createLearningPath("Python", "coding", 5);

// Branch to explore
const branch = eduAGI.branchLearningPath(path.id, "Web Dev", "Learn Flask");

// Get AI help
const help = await eduAGI.getAIAssistance(me.id, 'coding', "How do loops work?");
```

### For Teachers:
```typescript
// Create classroom
const classroom = eduAGI.createClassroom("AP Comp Sci", teacherId);

// Add students
eduAGI.addStudentToClassroom(classroom.id, student.id);

// Create team project
const project = eduAGI.createCollaborationProject(
  classroom.id,
  "Build a Calculator",
  "Team coding project",
  [student1, student2, student3]
);

// Track progress
const progress = eduAGI.getClassroomProgress(classroom);
```
