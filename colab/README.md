# Google Colab — Packest 5 (Alpha/Omega)

This folder contains a ready-to-run Google Colab notebook that installs and smoke-tests **5 core SDK packages** from this repository:

1. `packages/medina-alpha-sdk`
2. `packages/medina-organism-sdk`
3. `packages/medina-callable-sdk`
4. `packages/medina-research-lab-cli`
5. `packages/solus-ai-sdk`

## Open in Colab

- Public repo link (main branch):
  `https://colab.research.google.com/github/FreddyCreates/Medina-Memory-More-AI/blob/main/colab/MEDINA_PACKEST_5_ALPHA_OMEGA.ipynb`

If you're running a fork or a branch, replace `FreddyCreates/Medina-Memory-More-AI` and `main` in the URL.

## Notes

- For private forks, clone via HTTPS with a GitHub token (Colab will prompt for input if you use `getpass` in the notebook).
- The notebook uses `pip install -e` (editable installs) so you can iterate on code directly inside Colab.
