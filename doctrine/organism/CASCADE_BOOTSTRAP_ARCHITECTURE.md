### When to Package

Package when:
1. **Stable** → No more breaking changes
2. **Reusable** → Can be used by multiple products
3. **Self-contained** → Works independently
4. **Documented** → Clear API

### What to Package

- **Memory systems** → @medina/memory-temple-sdk
- **Intelligence routing** → @medina/three-hearts-sdk
- **Communication** → @medina/cpl-protocol-sdk
- **Security gates** → @medina/gate-security-sdk

### How to Package

```bash
#!/bin/bash
# Cascade bootstrap: Start core, then products, then services

echo "🌊 CASCADE BOOTSTRAP"

    echo "→ Bootstrapping core organism..."
    bash scripts/bootstrap-ai-workforce.sh
fi

# Level 1: Check all products
echo "→ Checking external products..."
for product in deployments/*/; do
    if [ -f "$product/bootstrap.sh" ]; then
        echo "  → Product: $(basename $product)"
        if ! check_product_running "$product"; then
            echo "    → Bootstrapping..."
            cd "$product" && ./bootstrap.sh
        else
            echo "    ✓ Already running"
        fi
    fi
done

echo "✓ CASCADE COMPLETE"
```
