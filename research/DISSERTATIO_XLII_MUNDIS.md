## I. INTRODUCTIO

### 1.1 The App Fallacy

The conventional approach treats VR/AR as "applications"—programs that run,
display interfaces, and terminate. This fundamentally misunderstands what
immersive computing enables.

**Thesis:** VR/AR environments are not apps. They are **worlds**—complete
ontological territories where beings (human, AI, cyborg, undead) can exist.

### 1.2 World vs Application

| Aspect | Application | World |
|--------|-------------|-------|
| Duration | Session-based | Eternal |
| Inhabitants | Users | Beings |
| State | Transient | Persistent |
| Laws | Code rules | Governance laws |
| Identity | Account | Citizenship |
| Death | Logout | State transition |
