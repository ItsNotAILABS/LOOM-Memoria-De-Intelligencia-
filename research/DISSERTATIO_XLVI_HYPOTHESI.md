## III. FALSIFIABILITY AND EVIDENCE

### 3.1 What Makes a Language Hypothesis Scientific?

A language hypothesis is scientific iff:

1. **Falsifiable**: There exist conditions that would disprove it
2. **Testable**: We can implement and measure
3. **Predictive**: It predicts specific outcomes
4. **Refinable**: It can be improved based on evidence

### 3.2 Evidence Criteria

| Criterion | Description | Measurement |
|-----------|-------------|-------------|
| Expressiveness | Problems that should be easy ARE easy | Problem-language fit |
| Efficiency | Computations that should be fast ARE fast | Performance metrics |
| Intuition | Constructs that should feel natural DO | User experience |
| Coherence | Extensions that should compose DO | Composability |

### 3.3 Evaluation Formula

```
HYPOTHESIS_SCORE(L) = Σ weights[i] × criterion[i](L)

WHERE:
  weights = [1.0, φ, φ², φ³]  # φ-weighted
  criteria = [expressiveness, efficiency, intuition, coherence]
  
SUPPORTED(L) iff HYPOTHESIS_SCORE(L) ≥ φ⁻¹
```
